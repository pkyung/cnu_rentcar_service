# cnu_rentcar_service
본 프로젝트는 23년도 1학기 데이터베이스 수업 과제로 진행한 렌트카 서비스입니다. 렌트카 검색, 예약, 반납, 결제 등을 구현했고, 결제 후에는 메일로 결과가 전송됩니다.
